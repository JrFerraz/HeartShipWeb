Demuestra tu habilidad

Dispara varios tipos de municion

Utiliza los distintos tipos de municion para acabar con los enemigos, utilizarlos a tu favor o jugar con el entorno. Elige entre:

Laser normal: Derrota enemigos y vuela cajas de municion.
Municion de impacto: Haz lo anterior y aturde enemigos. Tambien puede empujar multiples objetos.
Laser continuo: Rayo que acaba con todo enemigo que toque. Tambien se usa para resolver puzles.

Corre, cubrete y para el tiempo

Utiliza los generadores de velocidad para correr mas rapido, poder hacer evasiones mas largas y mas frecuentes.

Utiliza los escudos para cubrirte. Manten un equilibrio entre el escudo que consumes y el que recuperas.

Puedes ralentizar el tiempo para pensar mejor como te vas a mover, apuntar mejor etc.

Encadena rachas y consigue la mayor puntuacion

Derrotar enemigos en racha te otorgará una bonificacion de puntos, tambien conseguiras mas dinero

Consulta tus maximas puntuaciones para ver las estadisticas de tus mejores partidas

Usa el entorno a tu favor

Dispara a cajas de municion en vez de cogerlas para generar una explosion en cadena.

Encierra a los enemigos entre muros para evitar problemas.

Mueve objetos como pinchos, o plataformas reflectantes, para tener una mejor estrategia.

Consigue dinero, compra objetos

Los enemigos soltaran unas gemas verdes al ser derrotados, cuanto mayor sea tu racha actual, mas dinero soltaran los enemigos

Puedes usar el dinero para comprar nueva municion, armas, escudos generadores...

Juega de distintas formas

Los objetos en HeartShip se generan sus atributos de forma aleatoria, esto hace que haya objetos mejores que otros y a la vez, que haya muchos estilos de juego

Consigue los objetos naranjas para ser imparable, o equipate con los blancos para enfrentarte a un verdadero reto

Busca tu estilo. Elige si quieres ser rapido y agil, o ser mas lento y hacer mas daño, jugar estrategico... o jugar con lo mejor de cada estilo.
